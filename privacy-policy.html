Privacy Policy for MultiStopNavigation

Last updated: 2025-01-01

MultiStopNavigation is developed and maintained by Trương Văn Phap.

1. Information Usage

  • Location (GPS): Used only for navigation functionality.
  • Microphone: Used only when the user activates voice input.
  • Camera: Used only to scan QR codes containing location information.
  • Local Storage: Used only to store navigation history for the current day.

2. Data Collection

The application does NOT collect or store personal data on external servers.

3. Third-Party Services

  • Google Maps API
  • OpenAI API

4. Data Sharing

No user data is sold or shared for advertising or tracking.

5. Security

All data is processed locally or transmitted securely via HTTPS.

6. User Control

Users may disable permissions at any time via system settings.
