Privacy Policy for MultiStopNavigation

Last updated: 2025-01-01

MultiStopNavigation is developed and maintained by Trương Văn Phap. This Privacy Policy explains how information is processed when you use the application.

1. Information We Process

The application processes the following types of information strictly to provide its core functionality:

  • Location (GPS): Used in real time to calculate routes, display navigation paths, and guide users to multiple destinations.
  • Microphone (Voice Input): Used only when the user manually activates voice input to issue navigation commands.
  • Camera: Used only to scan QR codes that contain location data or Google Maps links.
  • Local Storage: Used to store navigation history for the current day on the device.

The application does not access any data without explicit user permission.

2. Data Processing and Storage

MultiStopNavigation does not permanently store personal data on external servers.

  • Location data is processed in real time and is not stored remotely.
  • Voice input is processed temporarily to fulfill the user's request and is discarded immediately after processing.
  • Navigation history is stored locally on the user’s device and is automatically cleared according to application logic.

3. Third-Party Services

The application uses the following third-party services to operate correctly:

  • Google Maps API – for map display, route calculation, and navigation.
  • OpenAI API – to process user commands and generate navigation-related responses.

Data sent to these services is used solely to fulfill user-requested functionality. The developer does not use this data for advertising, profiling, or resale.

4. Data Sharing

The developer does not sell, rent, or trade user data.

Data is shared only with the above third-party service providers when required to perform app functionality.

5. Security

All data transmissions use secure HTTPS connections. Temporary data processed by the application is handled with appropriate security measures.

Voice input and location data are not retained after the requested operation is completed.

6. User Control

Users have full control over their data and permissions:

  • Permissions (Location, Microphone, Camera) can be enabled or disabled at any time via system settings.
  • The application can be uninstalled at any time, which removes all locally stored data.

7. Children’s Privacy

MultiStopNavigation is not intended for children under the age of 13. The application does not knowingly collect personal information from children.

8. Changes to This Policy

This Privacy Policy may be updated from time to time. Any changes will be reflected on this page with an updated revision date.
